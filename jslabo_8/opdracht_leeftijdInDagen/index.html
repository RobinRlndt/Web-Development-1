Welk formaat wordt er eigenlijk gebruikt om het Date object? Milliseconden?

Het Date object in JSON wordt standaard weergegeven in het ISO 8601-formaat, zoals "2025-04-03T12:41:00.000Z".

Wat gebeurt er als eenzelfde object meermaals voorkomt bij het opbouwen van de JSON tekst?

Als eenzelfde object meerdere keren voorkomt tijdens het opbouwen van een JSON-tekst, zal het object meerdere keren worden afgedrukt.
