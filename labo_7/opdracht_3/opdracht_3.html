De generator voorziet de cellen in deze rijen van een andere achtergrondkleur, zoek op waar dit precies staat in de gegenereerde CSS.

De achtergrondkleur van de cellen in de rijen wordt ingesteld door een CSS-regel die de pseudo-class nth-child gebruikt, zoals:

.prognose tbody tr:nth-child(even) { background-color: #f9f9f9; } voor even rijen en .prognose tbody tr:nth-child(odd) { background-color: #ffffff; } voor oneven rijen.


Voorzie in je oplossingen van de vorige opdracht enkele cellen met onbekende gegevens (bijv. een geboortedatum die je niet kent, of niet zeker van bent). Zorg ervoor dat je zowel op een even als een oneven rij zo'n cellen hebt. Geef deze cellen een class 'onbekend' en stop een regel in je CSS waardoor ze een rode achtergrondkleur krijgen.

CSS code van 'homepage.css' voor cellen met onbekende gegevens:

.onbekend { background-color: red; color: white; }

"onbekend" invoeren in een cel

Waarom heeft de volgende CSS regel: .onbekend {background-color:red;} geen effect als je deze toepast op een <td> cel in een <tr> rij met class="alt", maar wel in een <tr> rij zonder deze 'alt' class? Hint: het heeft te maken met de specificiteit van de regels.

Dit komt omdat de CSS-regel voor rijen met de alt class specifieker is.

.vrienden tbody tr.alt td { background-color: #C4B7B7; } heeft een hogere prioriteit dan .onbekend { background-color: red; }, omdat de eerste regel meer specifieke selectors heeft.
