Je kan inderdaad makkelijk een heel element in één klik verwijderen, ook kan je de inhoud aanpassen --> het is inderdaad een dynamisch HTML-document

Opdracht 3

Elke accordion section bestaat uit:

  • een hoofdsectie (header) --> H3 in het voorbeeld
  • en zijn verborgen paneel (inhoud) --> DIV, P-ELEMENT

Wanneer je een andere section open klikt, sluit de andere. Hiervoor wordt een script gebruik dat automatisch de stijl van een class-atribuut verandert waardoor het sluit of openklapt.



Zodra de pagina wordt geladen kan Jquery UI attributen toevoegen zoals "aria-expanded".

De Jquery UI kan de elementen in de DOM-tree dynamisch manipuleren.
